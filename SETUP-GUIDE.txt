## Files Created (4 items)

### 1. **netlify.toml**
   - Netlify configuration file
   - Tells Netlify where your functions are
   - Location: Root folder

### 2. **package.json**
   - Node.js dependencies
   - Includes `mongodb` driver
   - Location: Root folder

### 3. **Backend Functions (3 files in netlify/functions/)**

   a) **save-entry.js**
      - Saves band entries to MongoDB
      - Called when user clicks "Add Entry"
   
   b) **get-entries.js**
      - Fetches all entries for a user from MongoDB
      - Called when loading data
   
   c) **delete-entry.js**
      - Deletes an entry from MongoDB
      - Called when user clicks "Delete"

### 4. **index.html**
   - Your existing Event Band Sales app
   - Needs small updates (see Step 6 in README)

### 5. **README.md**
   - Complete setup instructions
   - Troubleshooting guide
